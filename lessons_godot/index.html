Godot Engine Lessons and Tutorials

Game development is complex and involves a wide variety of knowledge and skills. In order to build a modern game, you need a great deal of underlying technology before you can make the actual game itself. This is where the "game engine" comes in.
A game engine is a collection of tools and technologies designed to ease the process of game development by removing the need to reinvent the wheel for each new game project. This includes things like rendering (displaying images on the screen), physics, platform support (mobile, console, etc.), and many other features.

What is Godot?

Godot is a fully-featured modern game engine that's very good at both 2D and 3D. One of Godot's big advantages is that it is 100% free. It is open source (MIT License), which means that you will never pay anything for it, no matter what. On top of all that, Godot supports all the most popular platforms for games - PC, mobile, and even consoles!

All lessons are available on our YouTube channel, and we're also adding text-based versions of every lesson for those that prefer reading over watching. Read on to see what's available.

Note: Unless noted otherwise, all lessons below use Godot version 3.1.

Godot 101: Getting started with Godot Engine

If you've never used a game engine before, or if you're just new to Godot, this is the place to start. Each lesson will build on the last, introducing you to more of Godot's features. Some of the things you'll learn how to do:

  • Use the Godot Editor to design your game
  • Use Godot's node and scene system to design robust, reusable game objects
  • Write code to capture input and build complex behaviors
  • Build user interfaces to display information
  • Use techniques you can apply to your own game projects!

"Dodge the Creeps": Your First Godot Game

It's time to take the skills you learned in "Godot 101" above and apply them to making a real game! By the time you're done, you'll have a complete game that you can share with your friends. You'll also have the opportunity to customize and extend the game to make it your own. Some of the things you'll learn how to do:

  • Organize your project into manageable scenes
  • Spawn enemies at random locations
  • Use animations to make characters more dynamic
  • Use Timers to spawn enemies
  • Build a simple UI for ending and restarting the game
