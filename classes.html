Overview

At KidsCanCode we make computers fun! By looking "under the hood," kids learn that a computer is a powerful tool - one they can use to create amazing projects of their own. Imagine building your own website, creating a video game, or wow-ing your teacher with an amazing science project. Our classes lay the foundation for a positive relationship with technology.

All classes are available for ages 10-14.

Introduction to
Programming

This is our core program, introducing the idea of a programming language and what it means to tell a computer what to do. We'll cover basic interaction with the Python console using statements to manipulate numbers and text. Next, we'll move on to Turtles - creating our own graphics and introducing loops and functions. Finally, we'll conclude with covering the workings of the Raspberry Pi - how to set it up at home, and what can be done with it.
NOTE: Tuition includes the Raspberry Pi kit, which will be used for all class activities, and the child will take home at the end of the session!

Requirements: No previous experience is required.

Programming with
Minecraft

Minecraft is one of the hottest games on the market - and best of all, it's all about creativity. Whether your child is a complete newbie or a Minecrafting expert, this class will have something new to offer. Using the special educational Minecraft: Pi Edition, we'll explore how to control the Minecraft world through code. We'll brainstorm to ideas of things to build, and show off our own creations on the last day of class. And the fun doesn't stop there: after class ends, every student will be able to bring his/her Raspberry Pi home to continue the Minecraft adventure!
Learn how to:

  • Create complex structures in only a few lines of code
  • Build objects that move!
  • Import 3d models from the web


Requirements: No previous experience is required.

Getting Started
with Raspberry Pi

Coming soon
So you have a Raspberry Pi - now what?
This class is all about the power of the Raspberry Pi. It's not only a great general purpose computing platform, it's also a great way to learn about electronics and foster a do-it-yourself mentality. We'll cover the basics of Linux (the operating system of the Pi), experiment with its capabilities, and create our own circuits to control LEDs and sensors via Python programs.

Requirements: No previous experience is required.
